\doxysubsection{Detailed Description}
The \mbox{\hyperlink{class_g_w_1_1_c_o_r_e_1_1_g_logic}{G\+Logic}} Interface stores and allows safe swapping of user logic (including \mbox{\hyperlink{struct_g_w_1_1_g_event}{G\+Event}} handlers). It combines the lifetime tracking of a G\+Interface(via Proxy) with the flexibility of std\+::function.
