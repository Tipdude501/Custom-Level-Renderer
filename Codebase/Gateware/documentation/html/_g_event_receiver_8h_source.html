51  /*!
52  * Receives any & all events produced by an interface derived from GEventGenerator.
53  * Accetps an std::function<void()> as the callback function/lambda for flexibility.
54  * Use std::bind to supply your own arguments to your callback, the only restriction is the void return type.
55  * To receive more detailed information, embbed this Proxy in your callback and "Pop()" the most recent event.
56  * \code{.cpp}
57  * MyEventReceiver.Create( MyEventGenerator, [](MyEventReceiver&) {
58  * GEvent g;
59  * GEventGenerator::Events e; // substitute the generator type you care about
60  * GEventGenerator::EVENT_DATA d;
61  * if(+MyEventReceiver.Pop(g))
62  * if(+g.Read(e,d))
63  * if(e == GEventGenerator::Events::NON_EVENT) // substiute actual event
64  * std::cout << d.noData; // substitute actual data
65  * });
66  * \endcode
67  * \deprecated **Replaced by GW::CORE::GEventResponder.**
68  */
