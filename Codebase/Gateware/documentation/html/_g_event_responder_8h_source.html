45  //! A Proxy/Functor class that can be used to immediately respond to a GEventGenerator's events.
46  /*!
47  * The GEventResponder proxy stores and allows safe swapping of GEvent handlers.
48  * It will immediately react to any GEventGenerator it is Registered to that posts a GEvent.
49  * Essentially this is a specialized version of GLogic designed for conveinent interception of events.
50  * **NOTE: GEventResponder is a replacement interface to GEventReceiver which is deprecated.**
51  * \code{.cpp}
52  * GEventResponder observer;
53  * // Provide the logic for processing a GEvent you care about.
54  * observer.Create([](const GEvent& g) {
55  * GEventGenerator::Events e; // substitute the generator type you care about
56  * GEventGenerator::EVENT_DATA d; // substitute the event data you care about
57  * // Determine if the incoming event contains the data you are interested in.
58  * if (+g.Read(e,d)) // if you are interested in multiple events consider using a switch below
59  * if (e == GEventGenerator::Events::NON_EVENT) // substiute actual event
60  * std::cout << d.noData; // substitute actual data
61  * });
62  * someEventGenerator.Register(observer);
63  * \endcode
64  */
65  class GEventResponder final
